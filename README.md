# Prod-Pusher

> Internal file publishing tool for IRCC Web Operations.

Prod-Pusher is a portable desktop tool that lets the web team push staging files directly to the live external and internal (Connexion) servers — no manual FTP client needed.

## Download

Get the latest version from the [**Releases**](https://github.com/uditcic/prod-pusher/releases) page.

Download `Prod-Pusher.exe` and run it directly — no installation required.

## Features

- 🌐 **External publish** — push pages to the public IRCC live server
- 🛠️ **Internal publish** — push pages to Connexion (intranet)
- 🔒 **Lock detection** — warns before overwriting pages locked by another coder
- 🧪 **Dry run** — preview what would be uploaded without pushing anything
- 📋 **Publish history** — searchable log of all past sessions
- 🔔 **Auto-update alerts** — notified in-app when a new version is available

## Usage

1. Download the latest `Prod-Pusher.exe` from [Releases](https://github.com/uditcic/prod-pusher/releases)
2. Run the `.exe` — it opens a browser window automatically
3. Choose **External** or **Internal**, enter your FTP credentials, paste your staging URLs, and publish
